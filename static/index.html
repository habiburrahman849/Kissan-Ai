Technical Dual-Brain Architecture

How Kissan AI processes queries to provide contextually-accurate guidance.

Farmer memory profile database

No more repeating yourself. Kissan AI learns from past conversations to build a persistent farmer profile using Mem0 semantic cloud memory.

Crop Tracking: Automatically records crop planting dates, harvest dates, and acreage inside the SQLite profile backend.
Farmer History: Retains context on available equipment, soil types, and language preference between visits.
Implicit Extraction: Extracts key farmer facts directly from regular chat messages without form-filling.
SQLite Backend Mem0 Cloud Integration Context Extraction
Local Fallback: If the cloud API keys are missing, the system utilizes local SQLite memory logs so the app works offline.

Pakistan-Grounded Knowledge Base

Grounded in Pakistan-specific documents rather than general worldwide advice, preventing incorrect fertilization and crop calendars.

Indexed Documents: Built-in vector parser reads official agricultural statistics, Punjab/Sindh crop calendars, and water consumption guides.
Localized Information: Answers specific questions like "wheat planting window in Multan" using local district guides.
Vector Seed Retriever: Performs fast local semantic similarity searches to supply relevant snippets directly to the Qwen LLM.
Official Pakistan PDF Assets Semantic Vector Chunk Index Anti-Hallucination Guardrails

Weather-Aware Spray Safeguards

Pesticides and liquid fertilizers get washed away or scattered in bad weather. Kissan AI parses live weather variables to issue alert notices.

Wind Check: Recommends postponing chemical sprays if wind speed exceeds 12 km/h.
Precipitation Forecasts: Warns against immediate tube-well operation if local rainfall is expected within 24 hours.
OpenWeatherMap API: Accesses real-time weather stations in Multan, Faisalabad, Lahore, and Karachi.
